Noţiunea de graf orientat

Exemplu: G=(X,U), X={1,2,3}, U={(1,2), (2,1), (1,3), (3,2)}

Definiţie: Se numeşte graf orientat o pereche ordonată de mulţimi G=(X,U) unde: -X este o mulţime finită şi nevidă numită mulţimea vârfurilor (nodurilor) -U este o mulţime formată din perechi ordonate de elemente din X numită mulţimea arcelor

Orice arc u∈U va fi notat cu u=(x,y) şi spunem că x este extremitatea iniţială iar y este extremitatea finală a arcului u. Într-un graf orientat arcul (x,y) nu este totuna cu arcul (y,x).

Ca şi în cazul grafurilor neorientate, dacă există arcul u=(x,y) spunem că vârfurile x şi y sunt adiacente şi amândouă sunt incidente cu arcul u.

Conform definiţiei de mai sus sunt admise şi arce de forma (x,x), adică arce de la un nod la el însuşi. Aceste arce se numesc bucle. De asemenea sunt admise şi arce identice de forma (x,y). Dacă numărul arcelor identice nu depăşeşte numărul natural p, spunem că G este un p-graf. Conform acestor definiţii, grafurile tratate în continuare sunt 1-grafuri fără bucle.

Observăm că pentru grafurile orientate sunt două tipuri de arce incidente cu un vârf: arce ce intră şi arce ce ies din acel vârf. Vom defini deci două două grade distincte care să corespundă numărului arcelor din fiecare categorie.

Definiţie: Numim grad exterior al unui vârf x∈X, notat cu d+(x), numărul arcelor de forma (x,y)∈U, adică numărul arcelor ce ies din vârful x.

Definiţie: Numim grad interior al unui vârf x∈X, notat cu d-(x), numărul arcelor de forma (y,x)∈U, adică numărul arcelor care intră în vârful x.

De asemenea vom nota cu: - Γ+(x)={y∈X|(x,y)∈U} mulţimea succesorilor lui x - Γ-(x)={y∈X|(y,x)∈U} mulţimea predecesorilor lui x - ω+(x)={u=(x,y)|u∈U} mulţimea arcelor ce ies din x - ω-(x)={u=(y,x)|u∈U} mulţimea arcelor ce intră în x

De asemenea, se poate arăta că într-un graf orientat G cu mulţimea vârfurilor X şi m muchii are loc relaţia : ∑d+(x)=∑d-(x)=m.

Spre deosebire de cazul grafurilor neorientate, un lanţ într-un graf orientat se defineşte ca un şir de arce L=[u1,u2,...,uk] cu proprietatea că oricare două arce vecine ui, ui+1 au o extremitate comună. În definirea unui lanţ nu se ţine cont de orientarea arcelor.

Exemplu: Pentru graful din figura alăturată putem defini lanţurile: L1=(u1,u2,u3), L2=(u1,u4,u5,u3) ambele de extremităţi 1 şi 4.

Un drum într-un graf orientat G=(X,U) cu X=(x1,x2,...,xn) este un şir de vârfuri notat D=(xi0,...,xim) cu proprietatea că oricare două vârfuri succesive reprezintă arce în graf.

Exemplu: Pentru graful din figura alăturată, D1=(1,2,1,3) şi D2=(1,2,3) sunt drumuri în graf.

Aşadar, un drum poate fi privit ca un lanţ în care toate arcele au aceeaşi orientare dată de sensul de deplasare de la xi0 către xim.

Dacă vârfurile =(xi0,.....,xim) sunt distincte, drumul se numeşte elementar.

Exemplu: În exemplul anterior D2 este drum elementar iar D1 nu este elementar.

Dacă D=(xi0,.....,xim) este un drum cu xi0=xim şi arcele (xi0,xi1),...,(xim-1,xim) sunt distincte, drumul D se numeşte circuit.

Dacă toate vârfurile circuitului cu excepţia primului şi ultimului nod sunt distincte două câte două circuitul se numeşte elementar.

Exemplu: Pentru graful din figura alăturată D1=(1,2,3,4,5,6,4,1), D2=(1,2,3,4,1) şi D3=(4,5,6,4) sunt circuite dar numai D2 şi D3 sunt elementare.

Noţiunile de subgraf şi graf parţial se definesc ca şi în cazul grafurilor neorientate şi anume:
Definiţie: Se numeşte graf parţial al unui graf orientat G=(X,U) un graf orientat G1=(X,V) cu V⊆U (deci este chiar G sau se obţine din G prin suprimarea unor arce).
Se numeşte subgraf al unui graf orientat G=(X,U) un graf H=(Y,V), unde Y⊆X iar arcele lui V sunt toate arcele din U care au ambele extremităţi în mulţimea Y (deci este chiar G sau se obţine din G prin suprimarea anumitor vârfuri şi a arcelor incidente cu acestea). Se spune că H este indus sau generat de mulţimea de vârfuri Y.

Exemplu: Pentru graful din figura de mai sus, G1=(X,V) cu V={u1,u2,u3,u7,u6} este un graf parţial, iar H=(Y,V) cu Y={1,2,3,4} şi V={u1,u2,u3,u4} este subgraful lui G indus de mulţimea de vârfuri Y.

Definiţie: Un graf orientat este complet dacă oricare două vârfuri distincte sunt adiacente.

Spre deosebire de grafurile neorientate la care există un graf complet unic cu n vârfuri, pentru un n dat, putem construi mai multe grafuri orientate complete având numărul de vârfuri n. Acest lucru decurge din faptul că două vârfuri x,y sunt adiacente în oricare din următoarele situaţii: există arcul (x,y), există arcul (y,x), există ambele arce (x,y), (y,x). Cum sunt C2n posibilităţi de a alege două vârfuri distincte iar pentru fiecare dintre acestea există trei situaţii ca ele să fie adiacente, rezultă că pentru un n dat sunt 3C2n grafuri complete cu n vârfuri.

Teoremă:Numărul total de grafuri orientate care se pot forma cu n noduri este 4C2n.

Metode de reprezentare a grafurilor orientate

Pentru exemplificarea metodelor de reprezentare specifice grafurilor orientate vom considera în cele ce urmează următorul graf: G=(X,U), n=4, m=5, X={1,2,3,4}, U={u1,u2,u3,u4,u5}.

Matricea de adiacenţă

Matricea de adiacenţă asociată unui graf orientat G=(X,U) se defineşte ca şi în cazul grafurilor neorientate: A=(aij)nxn cu aij= 1 dacă (xi,xj)∈U; 0 dacă (xi,xj)∉U;

Pentru graful considerat, matricea de adiacenţă are următoarea structură:

Linia Coloana
1 0 0 0 0
2 1 0 1 1
3 0 0 0 1
4 0 1 0 0
Observaţii:

1.Matricea de adiacenţă asociată unui graf orientat nu mai este neapărat simetrică ca în cazul grafurilor neorientate

2.∑nj=1aij reprezintă gradul exterior al nodului xi (suma elementelor de pe linia i)

3.∑nj=1aij reprezintă gradul interior al nodului xj (suma elementelor de pe coloana j)

Matricea vârfuri-arce (matricea de incidenţă)

Matricea vârfuri-arce sau matricea de incidenţă a grafului G=(X,U) cu n vârfuri şi m arce este o matrice B=(bij)nxm unde:

bij= 1 dacă xi este extremitatea iniţială a arcului uj -1 dacă xi este extremitatea finală a arcului uj 0 dacă xi nu este incident cu arcul uj

Vârful Muchia
- u1 u2 u3 u4 u5
1 -1 0 0 0 0
2 1 1 0 1 -1
3 0 -1 1 0 0
4 0 0 -1 -1 1
Observaţii:

1.Pe fiecare coloană j există exact două elemente nenule şi anume: unul egal cu 1 (ce corespunde extremităţii iniţiale a arcului uj) şi unul egal cu -1 (ce corespunde extremităţii finale a arcului uj)

2.Numărul de elemente 1 de pe linia i reprezintă gradul exterior al lui xi, iar numărul de elemente -1 de pe linia i indică gradul interior al lui xi; o linie cu toate elementele 0 corespunde unui vârf izolat.

Listele vecinilor

Pentru fiecare nod x∈G se construiesc două liste ale vecinilor săi: - L+(x) lista vecinilor succesori; conţine nodurile ce sunt extremităţi finale ale arcelor ce ies din nodul x; - L-(x) lista vecinilor predecesori; conţine nodurile ce sunt extremităţi iniţiale ale arcelor ce intră în nodul x;

Pentru graful considerat, cele două liste au următoarea structură:

Nodul x L+(x) L-(x)
1 ∅ 2
2 1,,3,4 4
3 4 2
4 2 2,3
Observaţii:

1.Această metodă de reprezentare se implementează cu ajutorul listelor alocate dinamic.

2. Se observă că L+(x) este ansamblul mulţimilor Γ+(x) pentru x={1,2,…,n}, iar L-(x) este ansamblul mulţimilor Γ-(x) pentru x={1,2,…,n}.

Vectorul muchiilor

Conform acestei metode de reprezentare, fiecare arc al grafului poate fi privit ca o înregistrare cu două componente, adică cele două noduri care constituie extremităţile arcului: - nod_in: nodul din care iese arcul (iniţial) - nod_sf: nodul în care intră arcul (de sfârşit)

Putem defini tipul înregitrare care descrie un arc astfel:

struct ARC{ int nod_in,nod_sf; };

Numărul real de elemente utilizate în vectorul v este numărul m de arce din graful G descris, adică elementele v[1], v[2],.....,v[m]. Fiecare element v[i] (i1...n) este un arc în graful G având extremităţile v[i].nod_in (extremitatea iniţială), respectiv v[i].nod_sf (extremitatea finală).
